Simple Store Locator Plugin for Wordpress with Custom post type, Custom Taxonomy, Flat Design and Settings Page.

== Description ==

With this plugin, you can easily put a store locator on your WordPress site. Its intuitive interface makes it the easiest plugin of its kind to use, and the flat design means youâ€™ll have a classy store locator that fits in perfectly with any WordPress theme.

= Back-end Features =

* Add Store
* Add Category with custom marker icon
* Custom marker image size
* Integrate with shortcode in any post or page
* Change Colors
* Support distances in Km or in Miles
* User Geolocation (require secure origin - https)
* Set a fallback geolocation data if the user's browser doesn't support geolocation (require secure origin - https).
* Change the User Marker Icon
* Change the Map Style
* Change the Radius Fill and Stroke Color
* Change the Directions Stroke Color and Opacity
* Enable/ Disable: Default UI, Map Type Control, Street View, Scroll Wheel
* Enable or Disable the Directions Panel
* Enable or Disable Cluster Option
* Change the Cluster Icon
* Enable or Disable the Filters
* Choose Different Map Language
* Enable or Disable the Google Maps Script to avoid conflicts 
* Change Filters Label Name

= Front-end Features =

* Ajax based front-end
* Choose Store Categories
* Set a Radius
* Show Stores List
* Automatically geocode your position (require secure origin - https)
* Search Store by: Title, Street, City, State, Zip Code
* Get Directions 
* Responsive Direction Panel
* Change the start point
* Drag user marker

= Multilingual Ready = 

* Tested with WPML

== Installation ==

How to install:

1. Search for "As Store Locator" via Plugins/Add New on your WordPress admin panel. OR Upload as-store-locator folder into `/wp-content/plugins/`
2. Activate Plugin
3. Customize the plugin from settings page
5. Add new store category with custom icon
6. Add new store
7. Add the shortcode [store_locator] in your post or page.
